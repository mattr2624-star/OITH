# OITH - One In The Hand
## Dating App Prototype

A clickable prototype for the OITH dating application, designed to demonstrate the core user experience of a quality-focused dating platform.

### Quick Start

Simply open `index.html` in any modern web browser to view the prototype.

**No server or installation required!**

### Features Demonstrated

1. **Onboarding Flow**
   - Splash screen with brand introduction
   - 3-step value proposition walkthrough
   - Account creation with basic info
   - Photo upload interface
   - Preference settings (interested in, looking for, age range, distance)
   - Subscription/payment screen ($10/month)

2. **Match Discovery**
   - One match at a time (core differentiator)
   - AI compatibility score display
   - Profile preview with bio, interests, and photos
   - Like/decline actions with swipe animations
   - Detailed profile view

3. **Match Success**
   - Animated match celebration modal
   - Direct path to conversation

4. **Chat Interface**
   - Real-time messaging simulation
   - Quick reply suggestions (icebreakers)
   - Message timestamps
   - Online status indicators

5. **Date Planning**
   - Date/time selection
   - AI-powered venue recommendations
   - Date confirmation flow

6. **Profile Management**
   - User stats (matches, dates, response rate)
   - Profile editing access
   - Match preferences
   - Settings and subscription management

### Navigation

- Use the **bottom navigation bar** to switch between Match, Chats, and Profile
- Use **back buttons** to return to previous screens
- On the match screen, you can:
  - **Swipe right** or press **→** to like
  - **Swipe left** or press **←** to decline
  - Tap the **info icon** to view full profile

### Design Philosophy

- **Warm, sophisticated palette** - Coral/terracotta accents with cream backgrounds
- **Typography** - Cormorant Garamond for display, DM Sans for body text
- **Mobile-first** - Designed as a phone app in a device frame
- **Smooth animations** - Transitions, swipes, and micro-interactions

### Technical Details

- Pure HTML, CSS, and JavaScript (no frameworks)
- Responsive design adapts to screen sizes
- CSS custom properties for easy theming
- Touch/swipe support for mobile devices
- Keyboard navigation support
