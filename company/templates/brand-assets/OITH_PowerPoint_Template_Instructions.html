How to Use This Template

  • Open PowerPoint and create a new blank presentation
  • Go to View → Slide Master to edit master layouts
  • Create custom color theme with OITH brand colors (see Brand Guide)
  • Install fonts: Cormorant Garamond and DM Sans from Google Fonts
  • Recreate each slide layout shown above in the Slide Master
  • Save as .potx template file for reuse
