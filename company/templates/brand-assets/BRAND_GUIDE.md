## Brand Overview

**OITH** (One In The Hand) is a premium dating app focused on quality connections over quantity.

## Logo

### Primary Logo
The OITH logo features a bird silhouette within an open hand, symbolizing "a bird in the hand is worth two in the bush" - the philosophy of valuing what's in front of you.

**Logo File:** `prototype/icons/icon.svg`

### Logo Usage
- Minimum size: 32px height
- Clear space: Equal to the height of the "O" in OITH
- Never stretch, rotate, or alter colors outside brand palette

## Typography

### Primary Fonts

#### Display Font: Cormorant Garamond
- **Usage:** Headlines, titles, logo text
- **Weights:** Regular (400), SemiBold (600), Bold (700)
- **Google Fonts:** `https://fonts.google.com/specimen/Cormorant+Garamond`

#### Body Font: DM Sans
- **Usage:** Body text, UI elements, buttons
- **Weights:** Regular (400), Medium (500), Bold (700)

## Microsoft Word Template Setup

### Step 1: Set Up Fonts
1. Download fonts from Google Fonts:
   - [Cormorant Garamond](https://fonts.google.com/specimen/Cormorant+Garamond)
   - [DM Sans](https://fonts.google.com/specimen/DM+Sans)
2. Install fonts on your system
3. In Word: Home â†’ Font â†’ Select fonts

### Step 2: Create Custom Theme Colors
1. Go to **Design** â†’ **Colors** â†’ **Customize Colors**
2. Set the following:
   - **Accent 1:** #C4584A (OITH Terracotta)
   - **Accent 2:** #A84639 (Terracotta Dark)
   - **Accent 3:** #4CAF50 (Success Green)
   - **Accent 4:** #FF9800 (Warning Orange)
   - **Accent 5:** #F5E6E3 (Terracotta Light)
   - **Accent 6:** #F8F4F2 (Warm Gray)
   - **Text/Background Dark 1:** #2D2926 (Charcoal)
   - **Text/Background Light 1:** #FDF9F7 (Warm White)
3. Save as "OITH Brand"

### Step 3: Create Styles
1. **Title Style:**
   - Font: Cormorant Garamond, 36pt, Bold
   - Color: #2D2926
   - Spacing: 24pt after

2. **Heading 1:**
   - Font: Cormorant Garamond, 28pt, SemiBold
   - Color: #2D2926
   - Spacing: 18pt after

3. **Heading 2:**
   - Font: DM Sans, 20pt, Bold
   - Color: #2D2926
   - Spacing: 12pt after

4. **Normal:**
   - Font: DM Sans, 11pt, Regular
   - Color: #2D2926
   - Line spacing: 1.5

### Step 4: Save as Template
1. File â†’ Save As
2. Select "Word Template (.dotx)"
3. Save as "OITH_Template.dotx"
